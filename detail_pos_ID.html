Point of Sales

Aplikasi Point of Sales berbasis website ini adalah project pribadi yang awalnya bertujuan membantu usaha milik keluarga. Namun saya juga berharap aplikasi POS berbasis website ini dapat membantu mikro, kecil dan menengah (UMKM), mulai dari pengelolaan produk, kontak supplier dan pelanggan, sampai pada pencatatan transaksi keuangan yang terjadi. Selain bertujuan untuk kebutuhan pribadi, aplikasi ini juga ditujukan untuk kebutuhan belajar untuk orang orang yang belajar pemrograman. Jadi saya tekankan diawal dan perlu dipahami bahwa, untuk saat ini aplikasi ini bersifat open source dan tidak untuk diperjual belikan.
Aplikasi ini dibangun menggunakan bahasa pemrograman php dengan framework Code Igniter 3, MySQL sebagai rdbms. Untuk antarmuka menggunakan template AdminLTE dengan bantuan css framework bootstrap v4.

Adapun fitur yang tersedia sebagai berikut :

  • Product Management
    • Unit & Category management
    • Product management
    • Stock (Good, damaged, opname stock) management, including stock mutation
  • Contact
    • Supplier Contact
    • Customer Contact
  • Transaction
    • Purchase Transaction
    • Sale Transaction
    • Expense and Revenue
    • Return to Supplier (exchange of goods and refunds)
    • Return from Customer (exchange of goods and refunds)
    • installment record (customer installments and installments to suppliers)
  • Setting
    • Shop setting
    • Bank account management
    • User management

Adapun user level yang tersedia diurutkan dari level tertinggi, sebagai berikut :

  • Administrator
    • Access permission : all features.
  • Owner
    • Access permission : All features, except user management and stock mutation.
  • Warehouse
    • Access permission : Product and stock related feature.
  • Cashier
    • Access permission : Sale, expense, revenue, return and customer installment transaction.
  • Purchasing admin
    • Access permission : Purchase, expense, revenue, return to supplier and installment record transaction.
