Evapotranspiration and effective discharge processing applications

This website-based application is a freelance project for a college student. This application aims to help observers process rain data in each watershed into the form of evaporation, transpiration and effective discharge using penman modifications as evapotranspiration calculation method, FJ MOCK and NRECA as effective discharge calculation methods.
This application is built using PHP as programming language with Code Igniter 3 framework, MySQL as rdbms, interfaces using templates AdminLTE with the help of css framework Bootstrap v4.
My responsibility in this project is to prepare flowchart for penman modifications and nreca calculation methods based on available material modules, prepare erd and dfd, and implement flowchart into lines of code.
